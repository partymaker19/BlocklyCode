Задача 1: Hello World

Соберите блоки так, чтобы в окне вывода появилась строка: Hello World!

Подсказка
Подсказка: используйте блок вывода текста и текстовый литерал.
Критерии звёзд:
  • ★★★ — оптимально (минимум блоков)
  • ★★ — хорошо
  • ★ — решение верное

Вывод в консоль:

Для вывода информации в консоль используйте следующие функции в зависимости от выбранного языка:

  • JavaScript:
    • console.log("текст") — вывод текста
    • console.log(число) — вывод числа
    • console.log("текст", переменная) — вывод текста и значения
    • console.log(`Значение: ${переменная}`) — шаблонные строки
  • Python:
    • print("текст") — вывод текста
    • print(число) — вывод числа
    • print("текст", переменная) — вывод текста и значения
    • print(f"Значение: {переменная}") — f-строки для форматирования
  • Lua:
    • print("текст") — вывод текста
    • print(число) — вывод числа
    • print("текст", переменная) — вывод текста и значения
    • print(string.format("Значение: %s", переменная)) — форматирование строк
  • PHP:
    • echo "текст" . PHP_EOL; — вывод текста
    • echo число . PHP_EOL; — вывод числа
    • echo "текст " . $переменная . PHP_EOL; — вывод текста и значения

Примечание: Результат вывода будет отображаться в окне вывода справа внизу.

Цикл for со счётчиком

Цикл for со счётчиком — это повторение действий, где переменная (например, i) последовательно принимает значения с заданным шагом.

Из чего состоит

  • Старт: с какого значения начинаем (например, 0)
  • Финиш/условие: до какого значения идём (например, до 10 включительно)
  • Шаг: как меняем счётчик (обычно +1)
  • Тело цикла: что делаем на каждом шаге (например, печатаем i)

Пример (1…5, каждое число с новой строки)

  • JavaScript 
    for (let i = 1; i <= 5; i += 1) {
  console.log(i);
}
  • Python 
    for i in range(1, 6, 1):
    print(i)
  • Lua 
    for i = 1, 5, 1 do
  print(i)
end
  • PHP 
    for ($i = 1; $i <= 5; $i++) {
  echo $i . PHP_EOL;
}

Частые ошибки

  • 1…4 вместо 1…5: забыли “включительно” (JS: <= 5, Python: range(1, 6))
  • Неправильный шаг: при шаге 2 получится 1, 3, 5
  • Печать не счётчика: внутри цикла нужно выводить именно переменную цикла

В Blockly это обычно блок “for” (Циклы) с полями FROM/TO/BY и переменной-счётчиком.

Цикл while

Цикл while повторяет действия, пока условие истинно. Он полезен, когда заранее неизвестно, сколько раз нужно повторить (например, «пока не угадали число»).

Как устроен

  • Условие: проверяется перед каждым шагом (если false — цикл заканчивается)
  • Тело цикла: действия, которые выполняются на каждом повторении
  • Изменение: внутри цикла нужно менять значение, влияющее на условие

Отличия от for

  • for: заранее известны старт/финиш/шаг (например, 1…5)
  • while: повторяем «пока …», количество повторов заранее неизвестно

Простой пример: повторяем, пока не ввели 5

  • JavaScript 
    let guess = Number(input());
while (guess !== 5) {
  console.log("Попробуйте ещё");
  guess = Number(input());
}
console.log("Готово!");
  • Python 
    guess = float(input())
while guess != 5:
    print("Попробуйте ещё")
    guess = float(input())
print("Готово!")
  • Lua 
    local guess = tonumber(input())
while guess ~= 5 do
  print("Попробуйте ещё")
  guess = tonumber(input())
end
print("Готово!")
  • PHP 
    $guess = (float) input();
while ($guess != 5) {
  echo "Попробуйте ещё" . PHP_EOL;
  $guess = (float) input();
}
echo "Готово!" . PHP_EOL;

Частые ошибки

  • Бесконечный цикл: забыли обновить переменную внутри цикла
  • Lua: «не равно» пишется как ~=
  • Ввод: строку нужно превратить в число (Number/float/tonumber)

В Blockly используйте блок while (Циклы) и обновляйте переменную guess внутри цикла.

Список (массив) и «для каждого элемента…» (forEach)

Список (или массив) — это набор значений в одном месте. Например: [1, 2, 3]. У списка есть порядок: у каждого элемента есть номер (индекс).

Зачем нужны списки

  • хранить много однотипных значений (числа, слова) вместе
  • перебирать элементы по одному и что-то с ними делать
  • считать сумму, количество, находить минимум/максимум

Что такое forEach

forEach означает «для каждого элемента списка». В нашем интерфейсе этот блок называется «для каждого элемента k в списке» — это одно и то же по смыслу. Он берёт элементы списка по очереди и внутри цикла даёт вам текущий элемент (например, k или item).

В примере JavaScript показан современный способ перебора: list.forEach(...). Blockly может сгенерировать обычный цикл for — по смыслу это то же самое.

Пример: вывести элементы и посчитать сумму

  • JavaScript 
    const list = [1, 2, 3, 4, 5];
let sum = 0;
list.forEach((k) => {
  console.log(k);
  sum = sum + k;
});
console.log(sum);
  • Python 
    list = [1, 2, 3, 4, 5]
sum = 0
for k in list:
    print(k)
    sum = sum + k
print(sum)
  • Lua 
    local list = {1, 2, 3, 4, 5}
local sum = 0
for _, k in ipairs(list) do
  print(k)
  sum = sum + k
end
print(sum)
  • PHP 
    $list = [1, 2, 3, 4, 5];
$sum = 0;
foreach ($list as $k) {
  echo $k . PHP_EOL;
  $sum = $sum + $k;
}
echo $sum . PHP_EOL;

В задаче 14: создайте list (или numbers), затем используйте «для каждого элемента … в списке» и переменную-накопитель sum.

Подсписок (sublist)

Подсписок — это часть списка. Например, из списка [1..10] можно взять подсписок с элементами 3, 4, 5, 6, 7.

Как сделать в Blockly

  • Создайте список 1…10 и сохраните его в переменную list
  • Возьмите подсписок блоком «взять подсписок» (Lists → get sub-list) и сохраните в sub
  • Сделайте «для каждого элемента k в списке» по sub и выводите k

Подсписок удобен, когда нужна только часть элементов: тогда не приходится писать условия внутри цикла.

Примеры в языках

В JavaScript индексация с 0, а второй параметр slice не включается. Например, чтобы взять элементы со 2-го по 4-й, можно сделать slice(1, 4).

  • JavaScript: .slice(start, end) 
    const list = [10, 20, 30, 40, 50, 60];
const sub = list.slice(1, 4);
sub.forEach((k) => console.log(k));
    start включается, end не включается.
  • Python: list[start:end] 
    list = [10, 20, 30, 40, 50, 60]
sub = list[1:4]
for k in sub:
    print(k)
    start включается, end не включается.
  • Lua: собрать подсписок вручную 
    local list = {10,20,30,40,50,60}
local sub = {}
for i = 2, 4 do
  table.insert(sub, list[i])
end
for _, k in ipairs(sub) do
  print(k)
end
    В Lua индексация обычно начинается с 1.
  • PHP: array_slice 
    $list = [10,20,30,40,50,60];
$sub = array_slice($list, 1, 3);
foreach ($sub as $k) {
  echo $k . PHP_EOL;
}
    В PHP array_slice: offset с 0 и length — сколько элементов.

Альтернатива без подсписка: можно перебрать индексы 3…7 и печатать list[i].

Фильтрация списка

Фильтрация — это когда мы перебираем элементы списка и оставляем только те, которые подходят под условие. Например: “только чётные числа”.

Как сделать

  • Переберите list блоком «для каждого элемента k в списке»
  • Внутри цикла сделайте if с проверкой чётности: k % 2 == 0
  • Если условие истинно — выводите k и добавляйте его в sum

Важно: фильтрация не требует отдельного “подсписка” — можно просто печатать подходящие элементы внутри if.

Примеры в языках

В JavaScript можно писать по-разному: Blockly может сгенерировать цикл for, а в примерах ниже показан вариант с forEach. Результат одинаковый.

  • JavaScript 
    const list = [3, 4, 7, 12, 15];
let sum = 0;
list.forEach((k) => {
  if (k % 2 === 0) {
    console.log(k);
    sum = sum + k;
  }
});
console.log(sum);
  • Python 
    list = [3, 4, 7, 12, 15]
sum = 0
for k in list:
    if k % 2 == 0:
        print(k)
        sum = sum + k
print(sum)
  • Lua 
    local list = {3,4,7,12,15}
local sum = 0
for _, k in ipairs(list) do
  if k % 2 == 0 then
    print(k)
    sum = sum + k
  end
end
print(sum)
  • PHP 
    $list = [3, 4, 7, 12, 15];
$sum = 0;
foreach ($list as $k) {
  if ($k % 2 == 0) {
    echo $k . PHP_EOL;
    $sum = $sum + $k;
  }
}
echo $sum . PHP_EOL;

Минимум и максимум среди отфильтрованных

Частая задача: сначала отобрать элементы (например, чётные), а потом найти среди них минимум и максимум. Один удобный способ — собрать отдельный список evens, а затем найти min и max для этого списка.

Новые слова в коде

  • evens.push(k) — добавить элемент в конец массива evens (JavaScript).
  • Точка в evens.push означает «взять метод у объекта/массива»: у массива evens есть метод push.
  • Иногда генерация Blockly в JS использует Math.min.apply(null, evens) вместо Math.min(...evens). Это два способа «передать элементы массива как аргументы функции».
  • evens.append(k) — добавить элемент в конец списка evens (Python).
  • JavaScript 
    const list = [5, 2, 9, 4, 1, 8];
const evens = [];
list.forEach((k) => {
  if (k % 2 === 0) evens.push(k);
});
const min = Math.min(...evens);
const max = Math.max(...evens);
console.log(`min=${min}`);
console.log(`max=${max}`);
  • Python 
    list = [5, 2, 9, 4, 1, 8]
evens = []
for k in list:
    if k % 2 == 0:
        evens.append(k)
print("min=" + str(min(evens)))
print("max=" + str(max(evens)))

Типы данных:

Текст (строка) нужно оборачивать в кавычки, а числа — нет.

Примеры:

В разных языках названия могут немного отличаться, но идея одинаковая.

Основные типы:

string (строка): текст. Пишется в кавычках.

number (число): числа (и целые, и дробные).

float (дробное число): число с точкой, например 3.14.

bool/boolean: логический тип — true или false.

  • "Привет" — строка (текст)
  • 5 — число
  • 3.14 — дробное число
  • true — логическое значение (да/нет)
  • null — «пустое значение»
  • undefined — значение “не задано”
  • [1, 2, 3] — список (массив)
  • { name: "Ann" } — объект (набор свойств)

Примеры вывода:

  • console.log("Привет") — правильно
  • console.log(5) — правильно
  • console.log(Привет) — ошибка (нет кавычек)

Что такое переменная?

Переменная — это “коробка” с именем, в которой можно хранить значение (например, число). Это удобно, когда одно и то же значение нужно использовать много раз или менять в одном месте — тогда не надо искать и править число везде.

Примеры:

  • JavaScript: let myAge = 10
  • Python: myAge = 10
  • Lua: local myAge = 10
  • PHP: $myAge = 10;

JavaScript: var, let и const

В JavaScript есть несколько способов объявить переменную. В современных примерах чаще используют let и const.

  • var x = 1; — старый вариант, область видимости функция
  • let x = 1; — можно менять значение, область видимости блок
  • const x = 1; — нельзя переназначить, область видимости блок

Примечание: const запрещает переназначение переменной, но содержимое объекта/массива менять можно.

Что такое “область видимости”

Область видимости — это место в коде, где переменная “видна” и её можно использовать. Если переменная объявлена внутри некоторого участка кода, то снаружи этого участка её может не быть видно.

  • Function scope (область видимости функции): переменная видна внутри всей функции.
  • Block scope (область видимости блока): переменная видна только внутри фигурных скобок { ... } (например, внутри if / for).

Мини-пример

  • var (function scope) 
    function demo() {
  if (true) {
    var a = 1;
  }
  console.log(a);
}
    Переменная a объявлена внутри if, но var делает её видимой во всей функции demo.
  • let/const (block scope) 
    function demo() {
  if (true) {
    let b = 2;
  }
  console.log(b);
}
    Переменная b видна только внутри блока if. Снаружи блока b “не существует”.

Для новичков правило простое: в JavaScript почти всегда используйте const, а если нужно менять значение — let. var лучше избегать в новых примерах.

Конкатенация строк

Конкатенация — это соединение строк в одну строку.

Идея простая: берём кусочки текста и “склеиваем” их вместе.

Примеры:

  • JavaScript:
    • "Город: " + city — обычное сложение строк
    • `Счёт: ${score}` — шаблонная строка
  • Python:
    • "Город: " + city — сложение строк
    • f"Счёт: {score}" — f-строка (удобно вставлять переменные)
  • Lua:
    • "Город: " .. city — оператор .. склеивает строки
    • string.format("Счёт: %s", score) — форматирование строки
  • PHP:
    • "Город: " . $city — оператор . склеивает строки
    • "Счёт: {$score}" — интерполяция в двойных кавычках

Примечание: если “склеиваете” строку и число, иногда нужно сначала превратить число в строку.

Инкремент и декремент

Инкремент — это увеличение значения на 1, декремент — уменьшение на 1.

Примеры:

  • JavaScript:
    • score = score + 1 — увеличить на 1
    • score = score - 1 — уменьшить на 1
  • Python:
    • score = score + 1 — увеличить на 1
    • score = score - 1 — уменьшить на 1
  • Lua:
    • score = score + 1 — увеличить на 1
    • score = score - 1 — уменьшить на 1
  • PHP:
    • $score = $score + 1; — увеличить на 1
    • $score = $score - 1; — уменьшить на 1

В Blockly это обычно делается так: берём текущее значение переменной, прибавляем 1 и снова присваиваем в ту же переменную.

Условные конструкции (if/else)

Условие позволяет программе принимать решения: если выражение истинно (true) — выполняется один блок кода, если ложно (false) — другой или ничего.

Примеры:

  • JavaScript:
    • if (score >= 50) { console.log("Passed"); } — выполнить действие при условии
    • if (age >= 18) { ... } else { ... } — две ветки: если / иначе
  • Python: 
    • if score >= 50:
  print("Passed")
      — условие с отступами 
    • if age >= 18:
  ...
else:
  ...
      — ветка else
  • Lua: 
    • if score >= 50 then
  print("Passed")
end
      — if ... then ... end 
    • if age >= 18 then
  ...
else
  ...
end
      — ветка else
  • PHP:
    • if ($score >= 50) { echo "Passed" . PHP_EOL; } — выполнить действие при условии
    • if ($age >= 18) { ... } else { ... } — две ветки: если / иначе

Операторы сравнения

Операторы сравнения возвращают логическое значение: true или false.

  • > — больше
  • < — меньше
  • >= — больше или равно
  • <= — меньше или равно
  • == — сравнение: равно
  • != — сравнение: не равно
  • ~= — не равно (в Lua)

Важно: == — это сравнение, а = — присваивание (например, x = 5). А != означает “не равно” (в Lua — ~=).

Если нужно “ничего не делать” — просто оставьте ветку else пустой или не добавляйте её.
